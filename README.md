# Task 3: Factorial Calculation using Recursion

## 📖 Description
This project implements a recursive function to compute the factorial of a given number.  
It demonstrates the use of recursion in Java and includes handling of edge cases such as factorial of `0` and negative numbers.

---

## 🎯 Objectives
- Implement a recursive method to calculate factorial.
- Handle edge cases:
  - Factorial of `0` returns `1`.
  - Negative numbers throw an error message.
